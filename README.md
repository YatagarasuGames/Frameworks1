# Инструкция для запуска


## Сначала надо склонировать проект

`git clone https://github.com/YatagarasuGames/Frameworks1`

Потом для запуска проекта надо ввести команду
`dotnet run`

Проект будет запущен на порту: 54254

## Валидация
В качестве модели использовался игрок(ник и количество очков)

### Были валидированы поля

* ник проверяется на пустоту
* количество очков проверяется на пустоту и натуральность
